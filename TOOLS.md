# TOOLS.md - 工具笔记

*记录你的本地配置、API 信息、快捷命令等。助手的随身速查卡。*

---

## Skills 提供的工具

Skills 定义*如何*使用工具。这个文件是用于记录*你的*具体配置——你的设置独有的东西。

## 应该记录什么

比如：
- 相机名称和位置
- SSH 主机和别名
- TTS 首选语音
- 扬声器/房间名称
- 设备昵称
- 任何环境特定的配置

## 示例

```markdown
### 相机
- living-room → 主区域，180° 广角
- front-door → 入口，运动触发

### SSH
- home-server → 192.168.1.100, 用户: admin

### TTS
- 首选语音: "Nova"（温暖，略带英国口音）
- 默认扬声器: Kitchen HomePod

### Gmail
- 账号: your@gmail.com

### 常用命令
- 查天气: weather Beijing
- 查日历: gog cal today
```

## 为什么分开？

Skills 是共享的。你的设置是你自己的。把它们分开意味着你可以更新 skills 而不会丢失你的笔记，也可以分享 skills 而不会泄露你的基础设施。
