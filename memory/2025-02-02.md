### 安装方法

```bash
# 1. 克隆仓库
git clone https://github.com/moltbot/openclaw-skills.git /tmp/openclaw-skills

# 2. 手动复制技能到 Moltbot skills 目录
cp -r /tmp/openclaw-skills/skills/<author>/<skill-name> \
  /usr/local/lib/.nvm/versions/node/v22.17.0/lib/node_modules/moltbot/skills/
```

### 注意事项

- Moltbot CLI 的 `moltbot skills` 命令不支持 `install` 子命令
- 需要手动复制技能文件夹到 Moltbot 的 skills 目录
- 技能安装后会自动被 Moltbot 识别并显示在 `moltbot skills list` 中
- 部分技能需要安装额外的 CLI 工具或配置才能使用
